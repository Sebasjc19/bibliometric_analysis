Análisis bibliométrico sobre "Computational Thinking"

Este proyecto explora la producción científica en torno al pensamiento computacional, organizando y visualizando datos extraídos de más de 14,000 publicaciones.

1. Fuentes de Datos

Después de una extensa búsqueda de 20 bases de datos, se ha seleccionado 6 de ella como ACM, IEEE, Sage, ScienceDirect, Scopus y Taylor & Francis. Cada publicación se identificó y unificó utilizando claves únicas como DOI, ISBN e ISSN, garantizando una base de datos libre de duplicados.

2. Variables Analizadas

  • DOI, ISBN, ISSN: identificadores únicos para evitar duplicados.
  • DOI, journal y país: información clave sobre la relación entre las publicaciones con mas citaciones, el journal y el país de afiliación.
  • Titulo, primer autor, año, journal, publisher, tipo de publicación y base de datos: variables necesarias para analizar la cantidad de articulos según estas variables o ser ordenadas mediante métodos de ordenamiento.
  • Abstract, categorías y palabras clave: se analizaron categorías como "Habilidades", "Herramientas", entre otras, extrayendo palabras clave y sinónimos de los abstracts.

3. Herramientas Utilizadas

    • Python 3.12: versión más reciente del lenguaje de programación Python, utilizada para desarrollar el proyecto y ejecutar el código.
    • Pandas: librería de Python especializada en la manipulación y análisis de datos, ideal para trabajar con estructuras como DataFrames y Series.
    • Bibtexparser: librería en Python para analizar archivos BibTeX, permitiendo extraer y trabajar con metadatos de publicaciones académicas.
    • Sqlite3: base de datos relacional ligera integrada en Python, utilizada para almacenar y gestionar los datos del proyecto de forma eficiente.
    • Matplotlib: librería para la creación de gráficos estáticos, animados e interactivos, utilizada para visualizar los resultados del análisis bibliométrico.
    • Networkx: librería para la creación, manipulación y estudio de la estructura, dinámica y funciones de redes complejas, utilizada para crear grafos de relaciones entre publicaciones, journals, y país.
    • Flask: framework web ligero para Python, utilizado para desarrollar la interfaz web del proyecto y mostrar las visualizaciones de los análisis bibliométricos.

4. Instrucciones

En el inicio de la página se encontrará una barra de navegación con 3 opciones, inicio siendo esta pagina. En datos bibliométricos se encontrará los resultados del analisis de las publicaciones cientificas, como cantidad de publicaciones según una variable en especifica, palabras más frecuentes, etc. En métodos de ordenamiento se encontrarán graficas que representan el tiempo de ejecución de ordenamiento de todas las publicaciones cientificas según diversos métodos existentes en la actualidad. En estos dos ultimos apartados aparecerán los resultados los cuales puede expandir dando click en la imagen, o, si la imagen no aparece correctamente puede dar click en 'Volver a generar'
